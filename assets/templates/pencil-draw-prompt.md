# AI试卷分析 - Pencil 画图 Prompt

> 将以下内容复制粘贴到 codebuddy 对话框中，让它在已打开的 `AI试卷分析.pen` 文件中绘制所有页面。

---

请在当前打开的 `AI试卷分析.pen` 文件中绘制 AI试卷分析产品的 10 个页面原型。这是一个 K12 教育类 APP 的功能模块。

## 设计规范

- **设备尺寸**: iPhone 375×812（竖屏）
- **主色调**: #00b894（薄荷绿）
- **辅助色**: #3b82f6（蓝色）、#ef4444（红色/错误）、#f59e0b（橙色/警告）
- **背景色**: #f5f5f5（浅灰）
- **卡片背景**: #ffffff，圆角 16px，轻微阴影
- **字体**: 系统默认，标题 18-20px 加粗，正文 14px，辅助文字 12px 灰色
- **间距**: 页面左右 padding 16px，卡片间距 12px
- **底部标签栏**: 高度 83px（含安全区），4 个 tab 图标
- **顶部状态栏**: 高度 44px，含 Dynamic Island 区域
- **导航栏**: 高度 44px，白底

## 页面布局

请将 10 个页面从左到右排列在画布上，每个页面之间间隔 40px。每个页面是一个 375×812 的白色矩形框架。

## 绘制要求

1. 先调用 `get_editor_state` 获取当前画布状态
2. 调用 `get_guidelines` 加载可用的设计指南
3. 使用 `batch_design` 批量创建，每次最多 25 个操作
4. 使用 `spawn_agents` 让多个 agent 并行绘制不同页面以提高效率
5. 每完成一批页面后用 `get_screenshot` 验证效果
6. 全部完成后用 `export_nodes` 导出 PNG 到项目根目录下的 `原型截图/` 目录
